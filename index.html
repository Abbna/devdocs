Styledocco Generator for Shopware

The styledocco generator creates a library of HTML files from the shopware projects .less files to provide a slim and useful documentation for users and developers.

The shopware .less files contain documentation and examples inside comments that are generated into HTML files by the styledocco generator. The result contains the source code, the less styles and (if possible) a working example of the specific elements.

LESS Files

The shopware .less files are split into different parts:

all.less - includes all other less files
components.less - includes the shopware components folder
hacks.less - contains hacks and fixes
ie.less - contains internet explorer fixes
mixins.less - includes shopware the less mixin folder
modules.less - includes the shopware modules folder
variables.less - includes the shopware less variables folder

_components/ - shopware component files
_mixins/ - shopware less mixin function files
_modules/ - shopware module files
_variables/ - shopware variables files
